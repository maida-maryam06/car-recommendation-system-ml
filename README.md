# 🚗 AutoSense — Car Recommendation System

A complete machine learning project that recommends cars based on user preferences,
built as part of the Machine Learning course (AIC 301) at Bahria University, Lahore.

The system runs two full ML pipelines on two real-world Kaggle datasets, covering
every stage from raw data to a live web interface — unsupervised clustering,
feature selection, class imbalance handling, and five supervised classifiers
including two boosting algorithms.

## Highlights
- Trains on **2 Kaggle datasets** (UCI Car Evaluation + Craigslist 426K listings)
- Saves **13+ charts** automatically — confusion matrices, elbow curves, PCA variance, accuracy comparisons
- Live **Flask API** + dark-themed frontend with per-feature dropdowns and instant recommendations
